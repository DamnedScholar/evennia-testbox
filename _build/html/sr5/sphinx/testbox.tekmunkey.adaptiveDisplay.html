testbox.tekmunkey.adaptiveDisplay.adaptiveDisplayFunctions.boxTextAt(text='', lboxchar=' ', rboxchar=' ', paddingchar=' ', linewidth=78)[source]¶

Wraps the specified text and presents it inside a boxed display at the specified linewidth, or 78 columns by default. If lboxchar, rboxchar, and paddingchar are not specified, default characters are pulled from default_display_vars.py

Parameters:
  • text – {str} The string to box.
  • lboxchar – {str} The left-side boxing character. If omitted, defaults to default_display_vars.borderChar_Left
  • rboxchar – {str} The right-side boxing character. If omitted, defaults to default_display_vars.borderChar_Right
  • paddingchar – {str} The this_pad_string character that extends from the end of each wrapped line to the right-side boxing character. If omitted, defaults to default_display_vars.boxText_padding
  • linewidth – {int} The output display width, in characters or columns.
Returns:

{str} The specified text wrapped to fit inside the specified boxing characters. The entire boxed line fits inside the specified linewidth.

testbox.tekmunkey.adaptiveDisplay.adaptiveDisplayFunctions.getClientScreenWidth(player)[source]¶

Returns the screen width reported by NAWS to the ServerSession by a player’s client, or returns the minimum width value of 40, or returns the default value of 78 if the ServerSession or Client returns 0 or 1 (which would mean NAWS was unsupported or badly supported in the client).

If for any reason the player object has multiple sessions attached to it, this function will return the value reported by the last session in their sessions collection, or it will return the default or minimum if that last reported value requires it.

If for any reason the player object has no sessions attached to it, this function will always return the default value 78.

Parameters:player – {player} An instance of the player typeclass.
Returns:{int} The player’s screen width, as determined by NAWS or defaults.
